# Movie-Sales-Analysis

## Project Overview

This project focuses on analyzing movie sales data using Power BI to identify revenue trends, genre performance, and regional distribution patterns. The goal was to transform raw movie sales data into an interactive dashboard that helps understand business performance through visual storytelling and data-driven insights.

The dashboard enables users to explore sales metrics dynamically using filters and visual comparisons across multiple dimensions such as genre, year, and region.

🎯 Objectives
Analyze overall movie sales performance
Identify top revenue-generating movies
Compare genre-wise contribution to total sales
Understand yearly sales trends
Evaluate regional distribution of movie sales
Build an interactive dashboard for decision-making support

🛠️ Tools & Technologies Used
Power BI
Data Modeling
DAX (Data Analysis Expressions)
Data Cleaning & Transformation
Interactive Dashboard Design


📂 Dataset Description

The dataset includes structured movie sales information such as:
Movie Title
Genre
Sales Revenue
Release Year
Regional Sales Performance

These attributes were used to analyze trends and generate meaningful insights through visualization.

📈 Dashboard Features

The Power BI dashboard includes:

KPI cards summarizing overall sales performance
Genre-wise revenue comparison
Year-wise sales trend analysis
Regional performance breakdown
Interactive slicers for dynamic filtering
Clean and structured visual layout for better interpretation

🔍 Project Workflow

1️⃣ Data Preparation
Cleaned dataset for missing and inconsistent values
Standardized column formats
Prepared structured data for dashboard modeling
2️⃣ Data Modeling
Established relationships between tables
Created calculated measures using DAX
Organized fields for efficient visualization
3️⃣ Dashboard Development
Designed visual components including:
KPI summary indicators
Genre comparison charts
Time-series revenue analysis
Regional sales distribution visuals
Interactive filters for user-driven exploration

📊 Key Insights
A small group of movies contributed significantly to total revenue
Sales varied noticeably across different genres
Year-wise trends revealed peak-performing release periods
Regional comparisons highlighted differences in audience engagement
Interactive filtering allowed deeper exploration of performance metrics

🚀 Project Outcome

This project demonstrates how Power BI can be used to:
Convert raw datasets into interactive dashboards
Track performance metrics effectively
Compare category-level contributions
Identify trends across time and regions
Support data-driven decision-making through visualization
